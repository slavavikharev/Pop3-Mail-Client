Pop3 Mail Client
================

What is it?
-----------
pop3 - приложение для просмотра входящих сообщений

Supported servers
-----------------
+ GMail
+ Yandex.Mail
+ Mail.ru
+ Yahoo.com

How to run?
-----------
./pop3.py [options]

Options
-------
+ -s --save - сохранять файлы из письма
+ -h --help - страница справки
